# Project: Recursion

This project demonstrates recursive and iterative solutions in JavaScript for classic algorithm problems: **Fibonacci sequence** and **Merge Sort**.

## Features

- **Iterative Fibonacci:** Generates the first `n` Fibonacci numbers using a loop.
- **Recursive Fibonacci:** Generates the first `n` Fibonacci numbers using recursion.
- **Merge Sort:** Sorts an array recursively using divide-and-conquer methodology.

## Getting Started

### 1. Clone the Repository

```bash
git clone git@github.com:krig6/odin-recursion.git
cd odin-recursion
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Demo

```bash
npm run start
```

This executes the demo in `src/index.js` and prints results for `fibs`, `fibsRec`, and `mergeSort`
